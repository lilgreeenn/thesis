# Thesis Timer PWA

## 部署到 GitHub Pages（5分钟搞定）

1. 去 https://github.com/new 新建一个 repo，名字随意比如 `thesis-timer`
2. 把这个文件夹里所有文件上传进去（直接拖拽到 GitHub 网页）
3. 进 repo Settings → Pages → Source 选 `main` branch → Save
4. 等1分钟，访问 `https://你的用户名.github.io/thesis-timer/`
5. 手机 Safari 打开这个链接 → 底部分享 → 添加到主屏幕 ✓

## 文件清单
- index.html    主应用
- manifest.json PWA配置
- sw.js         Service Worker（离线支持）
- icon-192.png  App图标
- icon-512.png  App图标（大）
